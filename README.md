# NetWise

NetWise is a web-based net worth tracker application built with HTML, CSS, and JavaScript that allows users to add assets (savings, investments, property, vehicles) and liabilities (loans, credit cards, mortgages) to automatically calculate their total net worth (Assets - Liabilities) and determine their financial health status—whether they're financially stable or in a debt trap—providing a clear snapshot of their true financial position at a glance.

# Features

- Asset Management 
- Liability Tracking 
- Real-Time Net Worth Calculation
- Add/Remove Items
- Simple & Intuitive Interface 
- Responsive Design

# Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts
- CSS Libraries
